# Security Policy

## Supported Versions

Security fixes are currently provided for the active `0.1.x` release line of
`hermes-plugin-mnemon`.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

Please report suspected vulnerabilities privately to the project maintainers
using the repository's private security reporting channel, if available. If no
private reporting channel is configured, contact the maintainers through the
repository's published contact method and avoid disclosing details publicly
until the issue has been reviewed.
